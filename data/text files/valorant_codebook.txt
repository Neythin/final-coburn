Valorant Codebook

-------------------------------

headshots / discrete / raw count / Total headshots hit from all games
headshot_percentage / continuous / headshots over total hits / Percentage of total hit shots that are headshots
aces / discrete / raw count / Number of aces this act
clutches / discrete / raw count / Number of round clutches this act
flawless / discrete / raw count / Number of flawless rounds this act
first_bloods / discrete / raw count / Number of first bloods this act
kills / discrete / raw count / Total number of kills from all games
deaths / discrete / raw count / Total number of deaths from all games
assists / discrete / raw count / Total number of assists from all games
kd_ratio / continuous / kills per death / Kill/death ratio from this act
kills_round / continuous / kills per round / Average kills per round over all games this act
most_kills / discrete / raw count / Most kills ever gotten in one game
score_round / continuous / score per round / Average score per round over all games this act
wins / discrete / raw count / Total number of wins this act
win_percent / continuous / wins over total games played / Percentage of games that are wins
agent_1 / discrete / -- / Most played agent based on hours
agent_2 / discrete / -- / Second most played agent based on hours

gun1_legs / discrete / leg shots over total hits / Percentage of gun 1's total hit shots that are legshots
